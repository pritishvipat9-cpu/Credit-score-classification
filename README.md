# Credit Score Classification

## Project Overview
This project focuses on predicting customer credit score categories using financial and demographic information.

## Business Problem
Financial institutions use credit scores to assess customer creditworthiness and make lending decisions. Accurate classification helps reduce financial risk.

## Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Classification Modeling
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Results
Successfully classified customers into credit score categories using machine learning techniques.
